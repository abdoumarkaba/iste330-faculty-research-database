SETUP INSTRUCTIONS (Local MySQL)
=====================================

Prerequisites:
- MySQL Server installed and running
- Java JDK 8+ installed
- mysql-connector-java-8.0.21.jar (included in project root)

STEP 1 — Set up the database
------------------------------
1. Open MySQL Workbench or a terminal with MySQL access.
2. Log in as root with password "student".
3. Run the schema file:
      mysql -u root -pstudent < sql/schema.sql
   Or in Workbench: File > Open SQL Script > select sql/schema.sql > Execute.
4. This creates the database "faculty_research_db", all tables, and sample data.

STEP 2 — Verify database configuration
---------------------------------------
The database configuration is already set in src/main/resources/db.properties:
      host=localhost:3306
      user=root
      password=student

No changes needed unless your MySQL setup uses different credentials.

STEP 3 — Compile the application
---------------------------------
From the project root directory, run:

   Windows:
   javac -cp .;mysql-connector-java-8.0.21.jar src/*.java -d out

   Mac/Linux:
   javac -cp .:mysql-connector-java-8.0.21.jar src/*.java -d out

STEP 4 — Copy configuration to output directory
----------------------------------------------
Copy the database properties file to the output directory:

   Windows:
   copy src\main\resources\db.properties out\

   Mac/Linux:
   cp src/main/resources/db.properties out/

STEP 5 — Run the application
-----------------------------
   Windows:
   java -cp out;mysql-connector-java-8.0.21.jar Main

   Mac/Linux:
   java -cp out:mysql-connector-java-8.0.21.jar Main

=====================================
USING THE APPLICATION
=====================================

LOGIN SCREEN
- Select your user type: Faculty, Student, or Public.
- Faculty and Student: enter your numeric ID (sample IDs: faculty 1-3, students 1-3).
- Public: no ID needed, click Login directly.

FACULTY PANEL (faculty IDs 1, 2, or 3)
- Your keywords are shown on login.
- Insert Abstract: type text in the text area, choose type (book/speaking), click Insert.
- Update Abstract: enter the abstract_id to change, type new text, click Update.
- Delete Abstract: enter the abstract_id to remove, click Delete, confirm prompt.
- Search Students: type a keyword, click Search Students — matching students appear in the table.

STUDENT PANEL (student IDs 1, 2, or 3)
- Search Faculty: type a keyword (e.g. "AI", "Security"), click Search Faculty.
  Results show faculty name, building/office, and email.
- View All Faculty: click button to list all faculty without filtering.

PUBLIC PANEL (no login)
- Enter 1 keyword (e.g. "HCI"), click Search — shows matching students and faculty.
- View All Entries: shows every student and faculty member in the database.
